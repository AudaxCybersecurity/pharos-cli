# Pharos Roadmap

Pharos is intentionally focused on defensive visibility and evidence-ready asset discovery.

## v0.1.x - Stabilization

- Improve README and documentation
- Add screenshots and sanitized report examples
- Improve test coverage for classification logic
- Add Markdown export
- Improve HTML report layout
- Add clearer error messages for permission issues

## v0.2.x - Usability

- Auto subnet detection with `--auto`
- Network interface selection with `--interface`
- Better handling of multiple local interfaces
- More structured asset risk levels: Low, Medium, High
- IoT confidence score
- Optional Markdown report export

## v0.3.x - Reporting

- Executive summary section in HTML reports
- Asset category summary
- Risk distribution summary
- Evidence-ready report footer
- Sanitized sample reports

## v0.4.x - Packaging

- Docker image
- Prebuilt release artifacts
- Improved installation script
- Versioned releases

## Long-Term Ideas

- Watch mode for newly observed devices
- Passive observation mode
- Optional lightweight service enrichment for authorized internal use
- Integration-friendly output formats

## Out of Scope

Pharos will not become an exploitation framework, payload delivery framework, phishing toolkit, C2 framework, credential access tool, or offensive automation platform.
